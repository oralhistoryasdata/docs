Example CSV Files

OHD includes several example CSV files to help you understand the structure and format required for your own projects.

Metadata Example
demo-ohd-metadata.csv

Sample metadata file showing how to structure your collection information. This file demonstrates the required columns and data format for describing your oral history interviews, including interviewee information, dates, locations, subjects, and connections to transcript and media files.

Transcript Examples
armantrout-transcript-example.csv

Interview transcript with poet Rae Armantrout from the CTRL+Shift project. Shows the standard speaker/words/tags/timestamp structure for transcript data.

Filters / Visualization Vocabulary Example
filters.csv

An example of the filters.csv file used to define the controlled vocabularly used to tag interviews. These serve as the foundation of the svg-based, multi-colored visualizations of interviews by subject.

Tip: Click on any filename above to download the example CSV file. Use these as templates to understand the required structure for your own oral history data.
